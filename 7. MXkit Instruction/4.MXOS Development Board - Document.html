MXKit

In 2018, MXCHIP launched the MXKit series of development kits for IoT and intelligent hardware prototype development. It provides developers with an out-of-the-box smart hardware solution that makes it easy to validate users' software and features, enabling products to connect to cloud service platforms and mobile phones quickly and securely, reducing development cycles and bringing them to market quickly.

The MXKit development kit consists of four parts: MXKit-Base, MXKit-Core, MXKit-Arduino (optional), and MXKit-Sensor (optional). among them:The MXKit-Base board is a general-purpose development board with a rich peripheral interface that is compatible with all MXKit-Core boards.

  • The MXKit-Core board is an IOT access core board that includes the MXCHIP wireless communication module;
  • The MXKit-Sensor board is an expansion board that contains a wide range of application sensors to facilitate rapid development of various IOT applications;
  • MXKit-Arduino board adopts standard Arduino interface, compatible with any MCU development board with Arduino interface, and provides IOT access function for user products with MXKit-Core board.

EXT-AT3080

EXT-AT3080 is a wireless module development board based on AT firmware, which was launched by Shanghai Qingke (MXCHIP).The standard Arduino interface is provided to support the peripheral debugging function based on the Arduino interface while quickly adding wireless Wi-Fi communication to the legacy device master MCU.

Wireless Mode Fast EMW3080 has built-in AT Firmware, supports "Basic AT command", or "direct-connected cloud AT command". Through the direct connection to the cloud [AT Directive] (docs/AT/2. Instruction Description.md), you can quickly and securely connect to various cloud service platforms. Currently, the cloud platforms that support AT Direct Direct include: Alibaba Ilop Cloud, and Gome Cloud.
